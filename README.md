# Odin Recipes

Ky projekt është pjesë e kursit **The Odin Project**.  
Këtu do të praktikoj bazat e HTML duke ndërtuar një faqe me receta.

Në fund të projektit, do të kem demonstruar njohuri në:
- Strukturimin e faqeve me HTML
- Përdorimin e lidhjeve (links)
- Organizimin e projekteve me Git & GitHub
